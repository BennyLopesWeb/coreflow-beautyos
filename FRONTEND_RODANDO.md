## ✅ Status: Iniciado

O frontend Expo está rodando em background.

## 📱 Como Acessar

### 1️⃣ Mobile - Expo Go (Recomendado) 📱

**Melhor experiência!**

1. **Instale o Expo Go** no celular:
   - **iOS**: App Store → "Expo Go"
   - **Android**: Google Play → "Expo Go"

2. **Escaneie o QR Code**:
   - No terminal onde o Expo está rodando, você verá um **QR Code**
   - Abra o app **Expo Go**
   - Escaneie o QR Code
   - A aplicação abrirá no celular

**Vantagens:**
- ✅ Funciona perfeitamente
- ✅ Performance nativa
- ✅ Todas as funcionalidades
- ✅ Experiência real do app

---

### 2️⃣ Web 🌐

**URL**: http://localhost:8081

⚠️ **Nota**: O Expo Router tem limitações conhecidas no web. Se houver problemas, use o mobile.

---

### 3️⃣ Emulador 💻

No terminal do Expo, pressione:

- **`i`** → Abrir no iOS Simulator (requer Xcode)
- **`a`** → Abrir no Android Emulator (requer Android Studio)

---

## 🎮 Comandos no Terminal

Enquanto o Expo está rodando, você pode usar:

- **`r`** → Recarregar app
- **`m`** → Alternar menu de desenvolvimento
- **`j`** → Abrir debugger
- **`c`** → Limpar cache
- **`Ctrl+C`** → Parar servidor

## 🛑 Parar Frontend

Para parar o frontend:

```bash
pkill -f "expo start"
```

Ou pressione `Ctrl+C` no terminal onde está rodando.
