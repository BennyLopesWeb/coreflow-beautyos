# 🔍 Como Verificar Erros no Console

## 📋 Passo a Passo

### 1. Abrir DevTools
- Pressione **F12** no navegador
- Ou clique com botão direito → "Inspecionar"
- Ou **Cmd+Option+I** (Mac) / **Ctrl+Shift+I** (Windows/Linux)

### 2. Ir para a Aba Console
- Clique na aba **"Console"** no DevTools

### 3. Verificar Erros
Procure por mensagens em **vermelho** que indicam erros:
- `Error: ...`
- `TypeError: ...`
- `ReferenceError: ...`
- `Cannot read property ...`

### 4. Copiar Erros
- Clique com botão direito no erro
- Selecione "Copy" ou "Copiar"
- Cole aqui para eu ver

## 🎯 O Que Procurar

### Erros Comuns:

1. **"Cannot read property of undefined"**
   - Algum objeto está undefined
   - Precisa verificar o código

2. **"Module not found"**
   - Import incorreto
   - Arquivo não existe

3. **"Cannot convert undefined to object"**
   - Problema com SSR
   - AsyncStorage no servidor

4. **"Element type is invalid"**
   - Componente não exportado corretamente
   - Import incorreto

## 📸 Como Me Enviar o Erro

1. Abra o Console (F12)
2. Veja os erros em vermelho
3. Copie a mensagem completa
4. Envie para mim

Ou tire um print da tela do Console mostrando os erros.
